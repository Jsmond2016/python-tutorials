# 需求文档

写给前端转岗 Python 开发工程师的教程

## 项目目标

创建一个**专为前端开发工程师设计**的 Python 转岗教程，帮助学员快速掌握 Python 核心概念和 Web 开发技能。

## 目标读者

- 具备 1-3 年前端开发经验的工程师
- 熟悉 JavaScript/TypeScript、React/Vue 等前端技术
- 希望转岗或拓展到 Python 后端开发
- 对全栈开发感兴趣的前端开发者

## 核心特色

| 特色 | 说明 |
|------|------|
| 对比学习 | 每个知识点都提供 Python 与 JavaScript 的对比 |
| 实战导向 | 每章配有练习题+答案，阶段性实战项目 |
| 循序渐进 | 从基础语法到 Web 开发的完整学习路径 |
| 主流技术栈 | 基于 FastAPI、SQLAlchemy 等主流 Python 库 |

- 第6章：面向对象编程

- 第8章：异常处理与文件操作
- 第9章：异步编程基础
- 第10章：装饰器与上下文管理器

**阶段项目**：数据处理脚本

### 第三部分：Python Web 开发 (第 11-15 章 + 项目3)

**章节内容**：
- 第11章：Web 开发入门
- 第12章：FastAPI 快速上手
- 第13章：数据库操作
- 第14章：身份认证与授权
- 第15章：API 测试与文档

**阶段项目**：博客 API 系统

### 第四部分：进阶实战 (第 16-20 章 + 项目4)

**章节内容**：
- 第16章：WebSocket 实时通信

- 第19章：日志与监控
- 第20章：容器化与部署

**阶段项目**：全栈实战项目

## 内容要求

### 1. 对比学习
- 每个 Python 概念都要与 JavaScript 进行对比
- 使用表格展示语法差异
- 说明设计理念的异同

### 2. 练习题
- 每章必须包含练习题
- 基础题：巩固基本概念
- 进阶题：综合运用知识
- 挑战题：拓展思维
- 所有题目提供详细答案

### 3. 代码示例
- 所有代码示例都可以直接运行
- 包含详细的注释说明
- 提供输出结果示例

### 4. 实战项目
- 每个阶段项目都要有完整的需求说明
- 提供项目结构参考
- 包含核心代码实现
- 展示完整的代码示例

## 技术方案

### 文档框架
- **VitePress 1.6+**
- 支持 Markdown
- 支持代码高亮
- 支持搜索功能

### 包管理
- **pnpm** (更快速、更节省磁盘空间)

### 部署
- 可部署到 GitHub Pages
- 可部署到 Vercel
- 支持 Docker 容器化部署
